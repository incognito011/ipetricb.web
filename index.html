Summary: Latitude vs. X

The purpose of this project was to create web pages for visualizations from our previous weather analysis, examining the effects of latitude on weather in over 600 cities across the globe.

The weather data displayed on these pages was obtained on June 29, 2018 via OpenWeatherMap API. JSON API data was analyzed using Python libraries (pandas, datetime, numpy) and plotted with matplotlib.

Please visit the Comparison dashboard to view all plots and navigate to each analysis summary by clicking on the corresponding plot. To return to this page, simply click on the button.
